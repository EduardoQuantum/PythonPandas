Clase 04: Herramientas de Selección

El arte de recortar la realidad con Pandas

Autor/a

Eduardo Lencina

En esta clase aprenderemos a navegar y extraer información específica de nuestro DataFrame. Seleccionar no es solo “recortar”; es el proceso de aislar las variables (columnas) y los casos (filas) que realmente responden a nuestra pregunta de investigación.

1. Selección de Columnas (El eje vertical)

Cuando seleccionamos columnas, estamos eligiendo qué variables nos interesan de nuestra base de interés.

1.1 El método .filter()

Es la forma más elegante y explícita de elegir múltiples columnas. Permite mantener el código organizado, especialmente cuando la lista de variables es extensa.

# Selección de columnas específicas
columnas_interes = ['CH04', 'CH06', 'P21']

# Aplicamos el filtro y visualizamos
df_reducido = df.filter(items=columnas_interes)
print(df_reducido.head())
Lo que estamos haciendo
  1. Definimos una lista: Creamos ‘columnas_interes’ con los nombres técnicos.
  2. Filtramos: ‘.filter(items=…)’ genera una nueva tabla solo con esas columnas.
  3. Mostramos: Agregamos ‘print(df_reducido.head())’ para que el alumno pueda ver en la terminal cómo quedó su tabla recortada.

2. Selección de Filas (El eje horizontal)

Aquí decidimos qué casos o encuestados entran en nuestro análisis según condiciones lógicas.

2.1 El método .query()

Es la herramienta preferida para filtrar registros mediante una consulta lógica legible.

# Seleccionamos solo a las personas mayores de 18 años
adultos = df.query('CH06 >= 18')

# Verificamos el resultado
print(adultos['CH06'].min())
Lo que estamos haciendo
  1. Consulta: Le pedimos a la base los registros donde la edad (‘CH06’) sea mayor o igual a 18.
  2. Verificación: Al hacer ‘print(adultos[’CH06’].min())’, el alumno confirma que el valor más bajo en su nueva tabla es efectivamente 18.

3. Variantes: Selección Técnica (.loc e .iloc)

Además de ‘.query()’, existen dos métodos fundamentales para “cortar” la base de forma precisa.

3.1 Localización por etiquetas (.loc)

Se usa cuando queremos filtrar filas y columnas simultáneamente usando sus nombres.

# Seleccionar personas de sexo 1 (Varones) y solo ver su Edad
edades_varones = df.loc[df['CH04'] == 1, 'CH06']
print(edades_varones.head())

3.2 Localización por posición (.iloc)

Útil cuando no conocemos los nombres pero sí la ubicación (índice) de los datos.

# Traer las primeras 10 filas y las primeras 3 columnas
recorte_posicional = df.iloc[0:10, 0:3]
print(recorte_posicional)
Lo que estamos haciendo
  1. .loc: Trabaja con nombres. Filtramos filas (donde sexo es 1) y elegimos la columna ‘CH06’.
  2. .iloc: Trabaja con posiciones numéricas. El rango ‘0:10’ trae las filas de la 0 a la 9, y ‘0:3’ las primeras tres columnas.

4. Entendiendo la Sintaxis: Corchetes vs. Paréntesis

4.1 Los Corchetes [] (Acceso Directo)

Se utilizan para señalar o entrar a una parte del objeto. Piensa en ellos como el dedo que señala un cajón.

# Acceso directo a una sola columna
columna_sexo = df['CH04']
print(columna_sexo.unique())
Lo que estamos haciendo

Accedemos al contenido de ‘’CH04’’ y con ‘.unique()’ imprimimos los valores que contiene para asegurar que estamos en la variable correcta.

4.2 Los Paréntesis () (Acciones / Métodos)

Son el “disparador” de una acción. Siempre que veas paréntesis, Pandas está trabajando para darte un resultado nuevo.

# Ejecutamos la acción de calcular el promedio
print(df['CH06'].mean())

5. Resumen de Herramientas

Herramienta Sintaxis Propósito
Filtro Col ‘.filter(items=[’A’, ’B’])’ Elegir variables por nombre.
Filtro Filas ‘.query(’A > 10’)’ Elegir casos por lógica.
Precisión ‘.loc[filas, cols]’ Selección híbrida por nombres.
Posición ‘.iloc[0:5, 0:2]’ Selección por ubicación numérica.
