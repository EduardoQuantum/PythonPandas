1. Introducción (Texto Narrativo)

En esta primera sesión vamos a transformar nuestra computadora en un laboratorio de análisis de datos. No se trata solo de escribir código, sino de crear un Pipeline reproducible.

¡Atención! Observación Crítica

Antes de comenzar, asegúrate de estar parado en la carpeta raíz de tu proyecto (C:\CursoPandas). Si la ruta en tu terminal no coincide, los comandos siguientes fallarán.

Zona de Prompting

Si tienes dudas sobre tu versión de Python, puedes usar el siguiente prompt con tu IA de confianza:

“Actúa como experto en Windows. Estoy configurando un entorno virtual de Python y quiero saber si mi versión instalada es compatible con Pandas 2.0. ¿Cómo lo verifico?”

2. Configuración Técnica (Caja de Código)

Es fundamental aislar nuestras librerías para que el proyecto sea estable en el tiempo. Copia estos comandos y ejecútalos en tu terminal de VS Code:

— COMANDOS PARA LA TERMINAL —

1. Crear el entorno virtual

python -m venv venv

2. Activar el entorno (Windows)

venv

3. Instalar librerías necesarias

pip install pandas pyreadstat

## ¡Atención! Observación Crítica
Antes de comenzar, asegúrate de estar parado en la carpeta raíz de tu proyecto (`C:\CursoPandas`). Si la ruta en tu terminal no coincide, los comandos siguientes fallarán.

### Zona de Prompting
