Clase 01: El Ecosistema de Trabajo

Instalación de VS Code, Python y Librerías

Autor/a

Eduardo Lencina

En esta clase instalaremos las herramientas base que nos acompañarán durante todo el curso. El objetivo es que los estudiantes logren autonomia técnica. La IA es el asistente y el experto. Comencemos

1. El Editor: Visual Studio Code (VSC)

El editor es nuestro lienzo. Elegimos VSC porque es el estándar de la industria y tiene extensiones que nos facilitan la vida.

1.1 Instalación y Extensiones

  1. Descargá el instalador para Windows en code.visualstudio.com.
  2. Una vez instalado, ve al icono de cuadrados en la barra lateral izquierda (Extensions).
  3. Busca e instala: Python (de Microsoft) y Spanish Language Pack (interfaz en castellano).

2. El Motor: Python 3.x

Python es el lenguaje, pero necesitamos instalar el intérprete en Windows para que la computadora entienda nuestras órdenes.

Durante la instalación de Python

Es VITAL marcar la casilla que dice “Add Python to PATH” al inicio del instalador. Si no lo haces, VS Code no encontrará a Python y nada funcionará.

3. Preparando la carpeta y las Librerías

No instalamos cosas “al voleo”. Vamos a crear una carpeta organizada y un entorno virtual para que nuestro proyecto sea prolijo.

# --- CONFIGURACIÓN INICIAL EN LA TERMINAL ---

# 1. Crear nuestra carpeta de estudio
mkdir C:\CursoPandas
cd C:\CursoPandas

# 2. Crear el Entorno Virtual (nuestro laboratorio aislado)
python -m venv venv

# 3. Activar el entorno
venv\Scripts\activate
Lo que estamos haciendo
  1. mkdir: Crea la carpeta en el disco C. Es mejor no usar carpetas con espacios en el nombre.
  2. venv: Crea una copia de Python dentro de tu carpeta. Así, lo que instalemos aquí no romperá otras cosas de tu PC.
  3. activate: Enciende ese Python local. Notarás que aparece un (venv) al principio de tu línea de comandos.

4. Instalando las herramientas de datos

Ahora instalaremos Pandas y los “motores” para leer archivos de Excel y SPSS que usaremos en las próximas clases.

# --- INSTALACIÓN DE LIBRERÍAS ---

# Actualizamos el instalador
python -m pip install --upgrade pip

# Instalamos el trío fundamental
pip install pandas openpyxl pyreadstat
Lo que estamos haciendo
  1. pandas: Nuestra herramienta principal para manejar tablas.
  2. openpyxl: El motor que permite a Pandas leer y escribir archivos .xlsx.
  3. pyreadstat: El motor que permite a Pandas abrir bases de SPSS (.sav).

5. Mesa de Ayuda y Prompting

🆘 Mesa de Ayuda: “Python no se reconoce como comando”

Si al poner python en la terminal recibes un error, es probable que no hayas marcado la casilla “Add to PATH” durante la instalación. Solución: Vuelve a ejecutar el instalador de Python, elige “Modify” y asegúrate de marcar esa opción. Luego reinicia VS Code.

Zona de Prompting: “Actúa como experto en soporte técnico de Python. Estoy en Windows 11 intentando instalar pandas mediante pip dentro de un entorno virtual, pero recibo un error de ‘permisos’ o ‘acceso denegado’. ¿Cómo puedo solucionarlo?”
